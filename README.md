# かんたん公開セット（GitHub Pages）

このフォルダを GitHub にアップロードするだけで LP を公開できます。

## 手順（超シンプル）
1. GitHub ログイン → 右上「+」→ New repository
2. リポジトリ名: `kanjuku-lp`（任意） / Public を選択 → Create repository
3. **Add file → Upload files** でこのフォルダの中身（`index.html` など）をまとめてドラッグ&ドロップ
4. **Commit changes** をクリック
5. **Settings → Pages** → Source: `main` / Folder: `/ (root)` → Save
6. 数十秒後に公開URLが表示されます: `https://ユーザー名.github.io/リポジトリ名/`

## 変更ポイント
- 連絡先や文言は `index.html` を開いて直接編集してください。
- LINE 申込URL・Googleマップ埋め込みの差し替えも同様です。
